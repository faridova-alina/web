# 🐱 Кот-Симулятор

Интерактивная игра-симулятор кота на чистом **HTML5 + CSS3 + JavaScript (ES6+)**, с отрисовкой через **Canvas** и сохранением прогресса в **LocalStorage**. Без сборщиков и зависимостей.

## Запуск

Просто откройте `index.html` в браузере (двойной клик). Для гарантированной работы шрифтов можно запустить локальный сервер:

```bash
cd Valdis
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

Игра запускается по кнопке **▶ Играть** (нужно для активации звука — требование браузеров).

## Возможности

### Кот
- Процедурная отрисовка на Canvas: мягкое «нарисованное» тело, а не геометрические фигуры.
- Детальные **уши** с раковинами и пучками шерсти.
- Настоящие **глаза**: белки, радужка с градиентом, вертикальные кошачьи зрачки, блики, еле заметные **бровки**, веки — кот **моргает** и **закрывает глаза**, когда спит.
- По **6 усов** с каждой стороны.
- **Лапки** с пальчиками и подушечками (видны, когда лапа приподнята/в прыжке).
- Хвост со светлым кончиком, плавно прилегающий к телу.

### Анимации (отдельная кнопка/клавиша на каждое действие)
| Клавиша | Действие |
|--------|----------|
| `1` | Стоять (дыхание, моргание, осмотр) |
| `2` / `← →` | Ходить |
| `Shift` (+движение) | Бежать |
| `3` / `Пробел` | Прыжок |
| `4` | Танец |
| `5` | Красться |
| `6` | Спать (темнеет, глаза закрыты, значки ZZZ) |
| `7` | Кушать (идёт к столику с миской) |
| `8` | Охота (мышки + весёлая музыка) |

### Кастомизация
- Цвет **шерсти**, **пятен/живота** и **глаз** (палитры + произвольный цвет, комбинации).
- Узоры: однотонный, полоски, пятна, смокинг.
- Своё **имя** коту с возможностью смены (✏️).
- Кнопка «🎲 Случайно» и живое превью.

### Геймплей
- Шкалы **сытости**, **энергии** и **счастья**; кот **толстеет** при кормлении и **худеет**, когда голоден.
- **Ловля мышек** по комнате (выползают из норки).
- Мини-игра **«Рыбалка»** 🎣 — котик с удочкой ловит рыбок по дорожкам глубины (← → двигать крючок, `Пробел`/клик — подсечь).
- **Звуки**: процедурное мяуканье и мурчание (Web Audio), весёлая музыка во время охоты и рыбалки.
- **Несколько локаций**: 🛏️ Комната (кровать, норка, мусорка, окно с занавесками и деревьями), 🌳 Сад (деревья, цветы, бабочки, облака), 🎣 Озеро (рыбалка).
- Прогресс автоматически сохраняется в LocalStorage.

## Структура

```
index.html        — разметка и UI
css/styles.css     — оформление (glassmorphism, градиенты)
js/utils.js        — математика и работа с цветом
js/audio.js        — звуковой движок (Web Audio API)
js/storage.js      — сохранение в LocalStorage
js/cat.js          — отрисовка и анимация кота
js/world.js        — сцены, мышки, частицы, деревья
js/fishing.js      — мини-игра «Рыбалка»
js/game.js         — игровой цикл, ввод, состояние, UI
```

Управление мышью: клик по сцене — кот идёт туда и мяукает.
